# Placeholder Assets

This folder is intentionally reserved for **non-sensitive, placeholder visual
assets** that support the portfolio documentation — for example:

- Anonymized architecture renderings exported from the Mermaid diagrams in
  [`../../diagrams/`](../../diagrams/).
- Generic, watermark-free illustrations or wireframes drawn for portfolio use.

## What must NEVER go here

To respect confidentiality and the NDA covering the original project, this folder
must **not** contain:

- Screenshots of the live product or any client-facing UI.
- Any image containing real customer data, internal dashboards, or analytics.
- Logos, brand assets, or proprietary design files belonging to the client.
- Anything revealing endpoints, environment values, or internal terminology.

> If a visual is needed, prefer regenerating it from the Mermaid diagram sources
> so it stays provably free of confidential content.

_Placeholder file — keeps the folder under version control._
